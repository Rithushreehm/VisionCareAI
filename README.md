# VisionCartAI 🛒🤖

AI-powered product analysis system for e-commerce platforms.

## Overview

VisionCartAI uses Generative AI and Computer Vision to analyze product images and automatically generate:

- Product category
- Product description
- Product tags
- Brand identification (when visible)

The goal is to reduce manual product listing work for e-commerce sellers.

## Features

✅ Upload product image  
✅ AI-based image understanding  
✅ Automatic product description generation  
✅ Product category prediction  
✅ SEO-friendly product tags  
✅ Brand detection when visible  
✅ Django web interface
